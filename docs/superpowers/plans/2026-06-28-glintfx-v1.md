# glintfx v1 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Entregar a v1 da `glintfx` — uma lib C++ **drop-in** que une RmlUi (UI HTML/CSS) a um renderer GL3, exibindo um showcase de efeitos (glow, degradê, blur, sombra, mask) com texto, consumível por um app externo sem wirar GL/janela à mão.

**Architecture:** Fachada RAII (`glintfx::App`) com pImpl que esconde RmlUi/GLFW/GL. Quatro módulos internos: M0 Bootstrap (Rml init/context/documento), M1 Platform (janela/contexto GLFW + SystemInterface), M2 Render (wrap do `RenderInterface_GL3` do RmlUi), M3 Facade (a API pública). Deps reais linkadas via CMake FetchContent (pinadas); efeitos vêm do CSS.

**Tech Stack:** C++ (CMake ≥3.10), RmlUi v6.3 (@`2cd28864ae25ed345b70598751703a5433b12356`), gl3w (@`d5ba9340cdeb9154323817f5c87e5a5c377fdef7`), GLFW, OpenGL 3.3 core, FreeType. Testes sob Xvfb.

## Global Constraints

(Cada task herda implicitamente esta seção.)

- **Plataforma:** Linux x86-64 apenas.
- **Padrão C++:** alvo `cxx_std_23`; **piso C++17** (a API pública compila em C++17→23).
- **Licença:** `SPDX-License-Identifier: MPL-2.0` no topo de **todo** arquivo novo (`.hpp`/`.cpp` com `//`, `CMakeLists.txt` com `#`). © 2026 Petrus Silva Costa.
- **Idioma:** identificadores en-intl; comentários/doc-comments bilíngues (en, depois pt).
- **Encapsulamento:** **nenhum** tipo de RmlUi/GLFW/GL/gl3w pode aparecer em `include/glintfx/*.hpp` (usar pImpl). Verificável por grep.
- **Deps pinadas:** RmlUi tag/commit `2cd28864ae25ed345b70598751703a5433b12356`; gl3w `d5ba9340cdeb9154323817f5c87e5a5c377fdef7`. `examples/` (clones de estudo) **não** é a fonte de build — o build usa FetchContent.
- **Build flags do RmlUi:** `RMLUI_SAMPLES OFF`, `BUILD_SHARED_LIBS OFF`, `RMLUI_FONT_ENGINE freetype` (FreeType embutido como default; a troca por outra engine é runtime via `Rml::SetFontEngineInterface`, exposta em task futura — fora da v1).
- **Pasta da lib:** `glintfx/` na raiz do repo. Camada 0 (`src/`) intacta.
- **Testes de GL:** rodam sob `xvfb-run` (sem display real).

```cpp
// SPDX-License-Identifier: MPL-2.0
#include "window_glfw.hpp"
#include <GL/gl3w.h>
#include <GLFW/glfw3.h>
namespace glintfx {
WindowGlfw::~WindowGlfw() {
  if (win_) glfwDestroyWindow(win_);
  glfwTerminate();
}
bool WindowGlfw::create(const char* title, int w, int h) {
  if (!glfwInit()) return false;
  glfwWindowHint(GLFW_CONTEXT_VERSION_MAJOR, 3);
  glfwWindowHint(GLFW_CONTEXT_VERSION_MINOR, 3);
  glfwWindowHint(GLFW_OPENGL_PROFILE, GLFW_OPENGL_CORE_PROFILE);
  win_ = glfwCreateWindow(w, h, title, nullptr, nullptr);
  if (!win_) { glfwTerminate(); return false; }
  glfwMakeContextCurrent(win_);
  if (gl3wInit() != 0) { glfwDestroyWindow(win_); win_ = nullptr; glfwTerminate(); return false; }
  return true;
}
void WindowGlfw::make_current() { glfwMakeContextCurrent(win_); }
void WindowGlfw::swap() { glfwSwapBuffers(win_); }
void WindowGlfw::poll() { glfwPollEvents(); }
bool WindowGlfw::should_close() const { return win_ && glfwWindowShouldClose(win_); }
void WindowGlfw::size(int& w, int& h) const { glfwGetFramebufferSize(win_, &w, &h); }
}
```

```cpp
// SPDX-License-Identifier: MPL-2.0
#include "bootstrap.hpp"
#include "render_gl3.hpp"
#include "window_glfw.hpp"
#include "RmlUi_Platform_GLFW.h"
#include <RmlUi/Core.h>
namespace glintfx {
struct Bootstrap::Impl {
  SystemInterface_GLFW system;
  Rml::Context* ctx = nullptr;
  bool initialised = false;
};
Bootstrap::~Bootstrap() { shutdown(); }
bool Bootstrap::init(WindowGlfw& win, RenderGl3& render, int w, int h) {
  impl_ = new Impl();
  impl_->system.SetWindow(win.handle());
  Rml::SetSystemInterface(&impl_->system);
  Rml::SetRenderInterface(render.iface());
  if (!Rml::Initialise()) return false;
  impl_->initialised = true;
  impl_->ctx = Rml::CreateContext("main", Rml::Vector2i(w, h));
  return impl_->ctx != nullptr;
}
bool Bootstrap::load(const char* rml_path) {
  if (!impl_ || !impl_->ctx) return false;
  Rml::ElementDocument* doc = impl_->ctx->LoadDocument(rml_path);
  if (!doc) return false;
  doc->Show();
  return true;
}
Rml::Context* Bootstrap::context() { return impl_ ? impl_->ctx : nullptr; }
void Bootstrap::shutdown() {
  if (impl_) {
    if (impl_->initialised) { Rml::Shutdown(); impl_->initialised = false; }
    delete impl_; impl_ = nullptr;
  }
}
}
```

```cpp
#include <glintfx/glintfx.hpp>
#include "window_glfw.hpp"
#include "render_gl3.hpp"
#include "bootstrap.hpp"
#include <RmlUi/Core.h>
namespace glintfx {
const char* version() { return "0.1.0"; }
struct App::Impl {
  WindowGlfw window; RenderGl3 render; Bootstrap boot; int w=0,h=0; bool ok=false;
};
App::App(AppConfig cfg) : impl_(std::make_unique<Impl>()) {
  impl_->w = cfg.width; impl_->h = cfg.height;
  if (!impl_->window.create(cfg.title, cfg.width, cfg.height)) return;
  if (!impl_->render.init()) return;
  impl_->ok = impl_->boot.init(impl_->window, impl_->render, cfg.width, cfg.height);
}
App::~App() = default;
App::App(App&&) noexcept = default;
App& App::operator=(App&&) noexcept = default;
void App::load(const char* rml_path) { if (impl_->ok) impl_->boot.load(rml_path); }
bool App::running() const { return impl_->ok && !impl_->window.should_close(); }
void App::poll_events() { impl_->window.poll(); }
void App::update() { if (auto* c = impl_->boot.context()) c->Update(); }
void App::render() {
  int w,h; impl_->window.size(w,h);
  impl_->render.begin_frame(w,h);
  if (auto* c = impl_->boot.context()) c->Render();
  impl_->render.end_frame();
  impl_->window.swap();
}
void App::run() { while (running()) { poll_events(); update(); render(); } }
}
```
